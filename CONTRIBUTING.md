# Contributing to Singularity Civilization | 贡献指南

Thank you for your interest in contributing to the Singularity Civilization Manifesto!
感谢您有兴趣为《奇点文明》白皮书做出贡献！

## Ways to Contribute | 贡献方式

### 1. Content Improvements | 内容改进
- **Bug fixes**: Correct errors in concepts, facts, or citations
- **Clarifications**: Improve unclear explanations
- **Translations**: Translate to other languages
- **Examples**: Add real-world case studies

### 2. Community Building | 社区建设
- **Discussions**: Participate in GitHub Discussions
- **Promotion**: Share the manifesto in relevant communities
- **Events**: Organize local meetups or reading groups

### 3. Technical Contributions | 技术贡献
- **Illustrations**: Create diagrams and visualizations
- **Tools**: Build tools based on the MVP protocol
- **Implementations**: Implement DTSP or AIGDP frameworks

## Submission Guidelines | 提交指南

### For Content Changes | 内容修改
1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/your-feature`
3. **Make changes**: Follow the style guide below
4. **Test**: Ensure links and formatting work
5. **Commit**: Use clear commit messages
6. **Pull Request**: Describe your changes in detail

### Commit Message Format | 提交信息格式
```
[type]: [description] | [中文描述]

[optional body explaining the change]
```

Types:
- `fix`: Bug fix | 错误修正
- `feat`: New feature/content | 新功能/内容
- `docs`: Documentation changes | 文档修改
- `style`: Formatting changes | 格式修改
- `refactor`: Code/content restructuring | 重构
- `i18n`: Internationalization | 国际化

Example:
```
feat: Add section on memory inheritance | 添加记忆继承章节

Added detailed explanation of 30-year frozen period
and guardian decision mechanism.
```

## Style Guide | 风格指南

### Writing Style | 写作风格
- **Clear**: Use precise language
- **Bilingual**: Provide English and Chinese side by side
- **Citable**: Include sources for factual claims
- **Constructive**: Focus on solutions, not just problems

### Diagram Standards | 图表标准
- Format: SVG for scalability
- Labels: Bilingual (English/Chinese)
- Colors: Follow the color scheme in `assets/style-guide.md`
- Accessibility: Include alt text

### Terminology | 术语规范
Use the glossary terms consistently:
| English | 中文 | Definition |
|---------|------|------------|
| Second Civilization | 第二文明 | Silicon-based Agent civilization |
| Cross-Dimensional Sovereignty | 跨维度主权锚定 | RoT-based dual-dimension anchoring |
| AIGDP | AIGDP | Agent Labor GDP |

## Review Process | 审核流程

1. **Automated checks**: Link validation, format checking
2. **Community review**: 7-day discussion period
3. **Maintainer review**: Final approval by @qq53732
4. **Merge**: Squash and merge with clear message

## Code of Conduct | 行为准则

- **Respectful**: Treat all contributors with respect
- **Constructive**: Provide constructive feedback
- **Open**: Embrace diverse perspectives
- **Ethical**: Uphold the Carbon-Silicon Social Contract
